This repository provides basic usage instructions for the three servers used in the Baldridge lab at Washington University in St. Louis. Click each server to view detailed instructions.

- [HTCF](https://github.com/xwu35/baldridge_lab/blob/main/HTCF.md): for computationally intensive analyses
- [baldridge VM](https://github.com/xwu35/baldridge_lab/blob/main/baldridge_VM.md): for hosting Shiny apps
- [pathogen server](https://github.com/xwu35/baldridge_lab/blob/main/pathogen_server.md): for R-based analyses
